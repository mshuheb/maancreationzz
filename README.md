# 🎨 Maan Creationzz

A modern and responsive printing service website built using **React.js + Vite**.
Showcasing custom apparel and merchandise including T-shirts, hoodies, mugs, and corporate gifting solutions.

---

## 🚀 Live Website

👉 **Check out the live website here:**
🌐 https://www.maancreationzz.in/

---

## ✨ Features

* ⚡ Fast performance with Vite
* 📱 Fully responsive design (Mobile + Tablet + Desktop)
* 🎯 Hero carousel with dynamic slides
* 🛍️ Service showcase (T-shirts, Hoodies, Mugs, Corporate Gifting)
* 💬 Detailed product/service explanations
* 🎨 Clean and modern UI with hover effects

---

## 🛠️ Tech Stack

* React.js
* Vite
* CSS3 (Responsive Design)

---

## 📂 Project Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/maan-creationzz.git
cd maan-creationzz
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

---

## 📱 View on Mobile

Run with:

```bash
npm run dev -- --host
```

Then open the provided network URL on your mobile device.
